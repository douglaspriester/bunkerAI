REQUISITOS (instalar UMA VEZ com internet):
═══════════════════════════════════════════
  1. Python 3.10+       → https://python.org
  2. Ollama             → https://ollama.ai
  3. Modelo de IA       → No terminal: ollama pull dolphin3

  Apos isso, tudo funciona 100% offline para sempre.

COMO USAR:
══════════
  Mac:     Duplo-clique em INICIAR.command
  Windows: Duplo-clique em INICIAR.bat
  Linux:   No terminal: cd app && bash start.sh

  O browser abre automaticamente em http://localhost:8888

PARA DISTRIBUIR (pendrive/HD externo):
══════════════════════════════════════
  1. Copie esta pasta inteira para o pendrive
  2. No computador destino, instale Python + Ollama
  3. Rode: ollama pull dolphin3
  4. Duplo-clique em INICIAR

  A primeira execucao cria o ambiente virtual automaticamente.
  Depois disso, tudo roda sem internet.

MODELOS RECOMENDADOS:
════════════════════
  ollama pull dolphin3           Chat principal (uncensored, ~5GB)
  ollama pull gemma3:4b          Visao + webcam (compacto, ~3GB)
  ollama pull qwen2.5-coder:7b  Gerador de apps (~5GB)
