ESTRUTURA DA PASTA:
═══════════════════
  BunkerAI/
  ├── INICIAR.command    <- Mac: duplo-clique aqui
  ├── INICIAR.bat        <- Windows: duplo-clique aqui
  ├── INICIAR.sh         <- Linux: bash INICIAR.sh no terminal
  ├── LEIA-ME.txt        <- Voce esta aqui
  └── app/               <- Arquivos do sistema
      ├── server.py
      ├── models/        <- Modelos LLM (.gguf)
      ├── bin/
      │   ├── mac/       <- llama-server para macOS
      │   ├── win/       <- llama-server.exe para Windows
      │   └── linux/     <- llama-server para Linux
      ├── static/
      ├── data/
      └── venv/

COMO USAR (ZERO configuracao):
═══════════════════════════════
  Mac:     Duplo-clique em INICIAR.command
  Windows: Duplo-clique em INICIAR.bat
  Linux:   No terminal: bash INICIAR.sh

  O launcher detecta automaticamente:
    - Quantidade de RAM disponivel
    - Presenca de GPU (Apple Silicon, NVIDIA)
    - Melhor modelo LLM para o hardware
    - Inicia llama-server + BunkerAI

  O browser abre automaticamente em http://localhost:8888

MODELOS INCLUIDOS:
══════════════════
  Pacote Light (~1 GB total):
    - dolphin-1b-q4.gguf  — CPU leve, roda em qualquer PC

  Pacote Full (~7 GB total):
    - dolphin-1b-q4.gguf  — CPU leve (fallback)
    - dolphin-8b-q4.gguf  — GPU uncensored (principal)
    - kokoro TTS          — Sintetizador de voz

  Selecao automatica pelo launcher:
    - RAM < 8GB ou sem GPU  -> modelo 1B (CPU)
    - RAM >= 16GB ou GPU    -> modelo 8B (uncensored)

REQUISITOS:
═══════════
  - Python 3.10+ instalado no sistema
    (Windows: baixe em https://python.org)
    (Mac/Linux: geralmente ja esta instalado)

  - Nenhuma internet necessaria apos primeira instalacao

PARA GERAR ESTE PACOTE (desenvolvedores):
══════════════════════════════════════════
  # Pacote leve (~1 GB, so modelo CPU)
  python build_portable.py --light

  # Pacote completo (~7 GB, 3 modelos + TTS)
  python build_portable.py --full

  # Opcoes avancadas
  python build_portable.py --output MinhaPasta --skip-model
