// BUNKERAI v0.1.0 · offline survival OS

WHEN THE INTERNET DIES...

...you'll wish you had this on a USB stick.

A fully offline operating system that runs in your browser. Local LLM, RAG over everything, offline Wikipedia, offline maps, survival toolkit, mini desktop environment, and 40+ apps. Zero telemetry.

// the toolkit

⌬

Local AI

Chat, voice, vision, and image generation — all running on your machine via Ollama + llama.cpp. Uncensored models included.

☰

Offline Library

Wikipedia, iFixit, Project Gutenberg, Stack Overflow — bundled as Kiwix ZIMs. Searchable without a connection.

◎

Offline Maps

Leaflet + Protomaps PMTiles. GPS support. Markers for water, shelter, hazards, food, medical, supplies.

✚

Survival Toolkit

16 guides, 10 emergency protocols as decision trees, water purification, first aid, ration planner, plant database, celestial navigation.

▣

Mini Desktop

Draggable windows, taskbar, start menu. Notepad, word processor, spreadsheet, paint, terminal, file manager. Cyberpunk dark.

◈

Portable Monolith

Copy server.py + static/ to a USB stick. Run it anywhere with Python and Ollama. No build, no framework.

▷

Media & Games

8 HTML5 games, ROM emulator for GB / GBA / NES, EPUB reader, media player. Because morale matters too.

⌧

Zero Telemetry

Nothing leaves the box. Not a single external call after setup. Single SQLite file holds all state — one file to back up.

// the stack

Python 3.10+ · FastAPI
Vanilla HTML / CSS / JS
SQLite (FTS5 + vectors)
Ollama + llama.cpp
Leaflet + Protomaps
Kiwix ZIMs
faster-whisper · Piper · Kokoro
Stable Diffusion (local)

One 4500-line server.py. No node_modules. No Docker required. Built to still run in 10 years.

// get started

git clone https://github.com/douglaspriester/bunkerAI.git
cd bunkerAI
./start.sh      # or start.bat on Windows

Requires Python 3.10+ and Ollama. NVIDIA GPU 8GB+ recommended, CPU works but slower.
